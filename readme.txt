== Description ==

Hearty Slider is a free responsive WordPress plugin that can be used to create a slideshow and customize each slide using the 90+ plugin settings.

= Product Details =

[Hearty Slider](https://www.heartyplugins.com/wordpress-plugins/free-plugins/product/hearty-slider) is the free version of our Hearty Slider Pro plugin, so compared to the full version, it has limited features and settings. This plugin has 1 settings instance, 10 content items, simple widgets and shortcodes, so for multiple settings instances, unlimited content items and flexible widgets, [check out the full version](https://www.heartyplugins.com/wordpress-plugins/premium-plugins/product/hearty-slider-pro).

== Installation ==

1. After downloading the plugin, install it using the standard WordPress plugin installation. Access PLUGINS/ADD NEW and upload the plugin you want to install.

1. Once installed, you need to activate the plugin and find the settings for the plugin inside SETTINGS menu.

For more details regarding the installation and initial configuration of the plugin, check out our step by step easy to follow [tutorial](https://www.heartyplugins.com/how-to-install-and-set-up-a-plugin).
